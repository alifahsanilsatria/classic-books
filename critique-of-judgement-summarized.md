## Bukti dari Geometri: Mengapa Pengetahuan Sintetik A Priori Nyata

```
    PROPOSISI: "Jumlah sudut segitiga = 180°"
    ══════════════════════════════════════════

    Uji 1: Apakah ANALITIK?
    ┌──────────────────────────────────────────────────┐
    │  Konsep "segitiga" = bangun tertutup, tiga sisi, │
    │  tiga sudut.                                     │
    │                                                  │
    │  Apakah "jumlah sudut = 180°" terkandung         │
    │  di dalam definisi itu?                          │
    │                                                  │
    │  TIDAK — kita tidak bisa mendapatkannya hanya    │
    │  dari menganalisis konsep "segitiga."            │
    │                                                  │
    │  ──► Proposisi ini SINTETIK                      │
    └──────────────────────────────────────────────────┘

    Uji 2: Apakah A POSTERIORI?
    ┌──────────────────────────────────────────────────┐
    │  Bisakah pengalaman membantah proposisi ini?     │
    │                                                  │
    │  TIDAK — tidak ada pengalaman yang mungkin       │
    │  menunjukkan segitiga (Euclidean) dengan         │
    │  jumlah sudut ≠ 180°. Proposisi ini berlaku      │
    │  NISCAYA dan UNIVERSAL.                          │
    │                                                  │
    │  ──► Proposisi ini A PRIORI                      │
    └──────────────────────────────────────────────────┘

    Kesimpulan:
    ╔══════════════════════════════════════════════════╗
    ║  Proposisi geometri = SINTETIK + A PRIORI       ║
    ║                                                  ║
    ║  Ini membuktikan bahwa pengetahuan sintetik      ║
    ║  a priori itu nyata — bukan fantasi.             ║
    ║                                                  ║
    ║  Pertanyaan: BAGAIMANA ini mungkin?               ║
    ╚══════════════════════════════════════════════════╝
```

          │                                     │
          ▼                                     ▼
  ┌─────────────────────┐              ┌─────────────────────┐

  Universal ──► Partikular             Partikular ──► Universal

  ┌──────────────────────────────────────────────────────────────┐
  │                                                              │

  │  → diterapkan pada      │  → diterapkan pada pengalaman     │
  │    intuisi empiris      │    melalui KEHENDAK & TINDAKAN    │
  │  → oleh PEMAHAMAN      │  → oleh AKAL PRAKTIS              │
  │                         │                                    │
  │  Keduanya MEMBERIKAN atribut pada objek                     │
  └──────────────────────────────────────────────────────────────┘

  Mengapa penilaian rasa BUKAN determinatif?
  ┌──────────────────────────────────────────────────────────────┐
  │                                                              │
  │  1. KEINDAHAN BUKAN SIFAT/PROPERTI dari objek               │
  │     "indah" secara gramatikal = predikat                    │
  │     tapi BUKAN atribut yang menentukan objek                │
  │                                                              │
  │  2. Tidak menggunakan KONSEP DETERMINATE                    │
  │     (tidak ada definisi "keindahan" yang bisa               │
  │      diterapkan sebagai aturan)                             │
  │                                                              │
  │  3. Tidak MENENTUKAN objek                                  │
  │     (tidak memberikan atribut pada objek)                   │
  └──────────────────────────────────────────────────────────────┘
